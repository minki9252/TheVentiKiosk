# 🧋 TheVentiKiosk 🧋

Qt/C++로 구현한 더벤티 콘셉트의 데스크톱 키오스크 애플리케이션입니다. 매장/포장 선택부터 메뉴 탐색, 옵션 선택, 장바구니 관리, 쿠폰/포인트 입력, 카드 결제 시뮬레이션, 영수증 및 주문번호 발급까지 실제 키오스크의 주문 흐름을 단계별 화면으로 구성했습니다.

> 본 프로젝트는 포트폴리오 및 학습 목적의 비공식 키오스크 구현물입니다.

## ✅ 주요 기능

- 매장/포장 주문 시작 화면
- 카테고리별 메뉴 조회
  - 신메뉴, 커피, 버블티/티, 블렌디드, 베버리지, 에이드, 디저트
- 메뉴 이미지, 이름, 가격 표시
- 옵션 선택 모달
  - 사이즈, 원두, 샷, 시럽, 음료 변경, 토핑 등 옵션 그룹 관리
  - 옵션 추가 금액을 반영한 실시간 총액 계산
- 장바구니 기능
  - 선택 메뉴 추가
  - 수량 증가/감소
  - 개별 삭제 및 전체 비우기
  - 총 결제 금액 계산
- 결제 단계별 플로우
  - 쿠폰 적용 및 할인 금액 반영
  - 포인트/스탬프 적립용 전화번호 입력
  - 카드 결제 방식 선택 및 결제 진행 시뮬레이션
  - 영수증 출력 여부 선택
  - 주문번호 발급 및 자동 종료
- SQLite 기반 메뉴/옵션/주문 관련 테이블 구성
- JSON 메뉴 데이터를 DB에 초기 적재
- Qt Resource System을 활용한 이미지 리소스 관리

## 💻 실행 방법 💻

### Qt Creator 사용

1. Qt Creator에서 `TheVentiKiosk.pro` 파일을 엽니다.
2. 빌드 Kit를 선택합니다.
3. 전체 프로젝트를 빌드합니다.
   - `VentiCore`가 먼저 빌드되고, `VentiApp`이 이를 링크합니다.
4. `VentiApp`을 실행합니다.

### qmake 사용

```bash
qmake TheVentiKiosk.pro
make
```

Windows 환경에서는 사용하는 Qt/MinGW 또는 MSVC Kit에 따라 `mingw32-make`, `nmake` 등을 사용할 수 있습니다.

## 구현 포인트

### 1. Core/App 분리

`VentiCore`는 SQLite 연결, 테이블 생성, 메뉴/옵션 조회를 담당하는 라이브러리로 분리했습니다. `VentiApp`은 UI와 사용자 흐름에 집중하며, 메뉴 조회와 옵션 데이터는 `DatabaseManager`를 통해 가져옵니다.

### 2. 메뉴 데이터 초기화

앱 실행 시 `menus.json`의 카테고리와 메뉴 데이터를 읽어 SQLite 테이블에 적재합니다. 메뉴 목록 화면은 DB에서 카테고리별 상품을 조회해 이미지, 메뉴명, 가격을 표시합니다.

### 3. 시그널/슬롯 기반 주문 흐름

Qt의 signal/slot 구조를 활용해 화면 간 데이터를 전달합니다.

```text
메뉴 선택
→ 옵션 선택
→ 장바구니 추가
→ 결제 다이얼로그 실행
→ 쿠폰
→ 포인트
→ 카드 결제
→ 영수증/주문번호
→ 주문 완료
```

### 4. 옵션 동적 생성

옵션 모달은 DB에 저장된 옵션 그룹을 읽어 버튼을 동적으로 생성합니다. 각 옵션의 추가 금액을 버튼 속성으로 관리하고, 선택 상태가 바뀔 때마다 총액과 옵션 요약을 갱신합니다.

### 5. 결제 시뮬레이션

실제 PG 연동 대신 키오스크 결제 흐름을 표현하는 데 집중했습니다. 카드 선택 후 결제 대기 화면, 카운트다운, 카드 삽입, 로딩, 결제 완료/취소 흐름을 UI 상태 전환과 타이머로 구현했습니다.

## 데이터베이스 구성

앱 실행 시 다음과 같은 주요 테이블을 생성합니다.

- `CATEGORIES`: 메뉴 카테고리
- `MENU_INFO`: 메뉴명, 가격, 이미지 경로, 품절 여부
- `OPTION_GROUPS`: 옵션 그룹명 및 단일/다중 선택 여부
- `OPTIONS`: 옵션명 및 추가 금액
- `MEMBERS`: 회원 전화번호, 스탬프, 누적 결제 금액
- `ORDERS`: 주문 내역
- `ORDER_ITEMS`: 주문 상세 항목

## 주요 화면 흐름

| 단계 | 설명 |
| --- | --- |
| 시작 화면 | 배너 이미지와 주문 시작 버튼 표시 |
| 주문 방식 선택 | 매장/포장 선택 |
| 메뉴 선택 | 카테고리별 메뉴 리스트 표시 |
| 옵션 선택 | 메뉴별 옵션 선택 및 가격 반영 |
| 장바구니 | 수량 변경, 삭제, 총액 확인 |
| 쿠폰 | 쿠폰 번호 입력 및 할인 적용 |
| 포인트 | 전화번호 입력 후 스탬프 적립 |
| 결제 | 카드 선택, 결제 진행, 취소/시간초과 처리 |
| 완료 | 영수증 여부 선택 후 주문번호 발급 |

## 개선 예정 사항

- 실제 주문 데이터를 `ORDERS`, `ORDER_ITEMS` 테이블에 저장
- 관리자 화면에서 메뉴/품절/가격 수정 기능 추가
- 쿠폰 및 회원 정보를 DB 기반으로 검증
- 빌드 환경별 실행 가이드 보강
- 화면 캡처 이미지 추가
